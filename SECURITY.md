# Security Policy

This document outlines the security policy for the **Solid Hypervisor** open-source project.

## Reporting a Vulnerability

We take security seriously and appreciate your efforts to responsibly disclose any vulnerabilities you discover.

For **critical or high-severity security issues**, please report directly via email to:  
**open-siengine@siengine.com**

In your report, please include:
- A clear description of the vulnerability
- Steps to reproduce the issue
- The affected version(s) of Solid Hypervisor
- Any potential impact or proof-of-concept (if available)

We will acknowledge your report within 5 business days and work with you to address the issue promptly.

> **Note**: For non-security-related bugs or feature requests, please use the project's GitHub Issues instead.

Thank you for helping keep Solid Hypervisor secure!
